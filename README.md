## Ruby使用Socket处理Http请求

### How to use

terminal input:

```ruby
# you can install gem or load the file 
$ gem install yn_server
$ irb
$ require 'yn_server'
# use YNServer.start(port) to start the server, port defaults to 2000
$ YNServer.start
```
